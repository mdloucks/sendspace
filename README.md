# Sendspace

An app for climbers to record & share their progress.

### To generate types

```bash
flutter pub run supabase_codegen:generate_types --output lib/core/data/models/dto
```

### To run build_runner

```bash
flutter pub run build_runner build --delete-conflicting-outputs
```
